# 🏓 PingPong

A lightweight task manager. Threads, people, groups, due dates.

## Setup

```bash
git clone https://github.com/MarinaMancoridis/PingPong.git
cd PingPong
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Open [localhost:8000](http://localhost:8000).

## Install as app (optional)

Visit `localhost:8000` in Chrome → click the install icon in the address bar.
